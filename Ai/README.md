# AI Task Architect v1.1 

**AI Task Architect** to inteligentne narzędzie zbudowane w PHP, które pomaga programistom i project managerom okiełznać chaos w wymaganiach biznesowych. Aplikacja analizuje opis zadania i generuje techniczny plan działania (kroki), ocenia trudność oraz szacuje czas realizacji.

## Funkcje projektu:
* **Dwujęzyczność (PL/EN):** Interfejs i analizy dostępne w dwóch językach.
* **Integracja z OpenAI:** Możliwość podłączenia klucza API dla realnej analizy opartej na sztucznej inteligencji.
* **Demo Mode:** Wbudowana logika symulująca działanie AI (przydatne do prezentacji bez użycia tokenów).
* **Dynamiczna ocena:** System generuje stopień trudności oraz estymację czasu pracy.

## Technologia:
* **Backend:** PHP 8.x
* **Frontend:** HTML5 / CSS3 (Dark Mode UI)
* **Integracja:** OpenAI API (JSON)

## Jak uruchomić:
1. Sklonuj repozytorium do folderu `htdocs` (XAMPP).
2. Otwórz `ai_engine.php` i opcjonalnie wstaw swój `apiKey`.
3. Uruchom projekt w przeglądarce: `localhost/nazwa_folderu`.
